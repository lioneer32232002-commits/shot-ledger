## 部署程序（兩種都已設好）

```bash
# A. wrangler 直接部署（本機需 wrangler login；用乾淨的 git archive，避免帶入雜檔）
DEPLOY_DIR=$(mktemp -d) && git archive main | tar -x -C "$DEPLOY_DIR" \
  && cd "$DEPLOY_DIR" && wrangler pages deploy . --project-name shot-ledger --branch main
```
- B. 使用者之後可在 Cloudflare dashboard 把 Pages 專案連到 GitHub repo → push 即自動部署（flight-deck 模式）。
- ⚠️ `.wrangler/`、`.claude/` 已在 .gitignore——**不要**解除忽略（曾誤 commit wrangler cache，已重寫歷史移除）。
