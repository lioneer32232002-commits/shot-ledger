## 部署程序

```bash
# wrangler 直接部署（本機已 login；用乾淨 git archive）
DEPLOY_DIR=$(mktemp -d) && git archive main | tar -x -C "$DEPLOY_DIR" \
  && cd "$DEPLOY_DIR" && npx --yes wrangler pages deploy . --project-name shot-ledger --branch main
```
